== Description ==
A simple **Admin Notepad Plugin** for WordPress. It saves notes automatically and displays a widget on the dashboard.

== Features ==
✅ Auto-save notes while typing  
✅ Dashboard widget for quick access  
✅ Simple, clean, and lightweight  

== Screenshots ==
1. Admin Notepad in WordPress menu.
2. Notepad widget on the WordPress dashboard.

== Installation ==
1. Upload `admin-notepad` folder to `/wp-content/plugins/`
2. Activate the plugin in **WordPress > Plugins**.
3. Find **"Admin Notepad"** in the Admin Menu or Dashboard.
